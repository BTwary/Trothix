# Security Policy

## Supported Versions

| Version | Supported |
| :--- | :--- |
| 1.0.x | Yes |
| < 1.0.0 | No |

## Reporting a Vulnerability

Please report any security vulnerabilities by opening a confidential ticket or emailing the security team directly.

## Dependency Update Policy

Security-related dependencies are monitored weekly and updated automatically if security advisories are issued.
